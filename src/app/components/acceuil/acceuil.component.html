Location de serveurs

Créez, déployez et gérez des applications sur plusieurs clouds, localement et à la périphérie

Machine Virtuelle
Configurez votre machine virtuelle sur un serveur distant. Choisissez votre OS, la localisation, la RAM, le nombre de processeurs, l'espace de stockage et beaucoup d'autres paramètres.
SQL Database

Configurez votre base de données SQL sur un serveur distant. Choisissez le mode de compatibilité, la taille de mémoire, le niveaux de sécurité et beaucoup d'autres paramètres.

Serveur de Stockage

Configurez votre serveur de stockage sur un serveur distant. Choisissez le système de fichier, le type de RAID, la capacité de stockage et beaucoup d'autres paramètres.

Cosmos Database

Configurez votre base de données cosmos sur un serveur distant. Choisissez le niveau de cohérence, le nombre de régions, la taille, la performance et beaucoup d'autres paramètres.
